9-20. Многопоточность
Реализуйте вашу задачу в виде клиент/сервер.

Клиент - это диалоговое меню.

Сервер - выполняет расчеты.

Сервер работает в одном потоке и эмулирует длительные расчеты (в функции добавьте случайную паузу). 

Клиентов может быть много и каждый запускается в своем потоке.

Включите логирование в файл действий сервера.

В консоль выводите сообщения типа

"12:00:00 Клиент1: отправлен запрос на сортировку массива"
"12:01:01 Клиент2: сгенерированы данные"
"12:01:01 Клиент1: выполнена сортировка массива"
